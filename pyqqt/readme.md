# SLICER PRO · 朋友圈切图工具

> 一张图，切成九宫格｜长图切片｜高清导出｜防越界水印

![效果示例](https://img.cdn1.vip/i/69eda24250975_1777181250.webp)

## 这是什么

**朋友圈切图**（pyqqt）是一种常见玩法：把一张完整图片切成若干小块，再依次发到朋友圈，滑屏时就能拼回原图。  
本项目提供一个**在线切片工具**，支持：

- 自由拖拽**裁切框**（九宫格 / 三连排 / 自定义比例）
- 添加 **水印**（支持「无损反色」与「纯色」两种模式）
- **强制物理比例锁定** & **右下角预设死锁**（适合品牌角标）
- 一键导出 **高清 ZIP 包**（原图分辨率切片，无水印画质损失）

## 核心功能

| 功能 | 说明 |
|------|------|
| **切图预设** | 一键切换 3×3（九宫格） / 1×3（三连排） |
| **裁切框** | 白色半透明参考线，可缩放、移动，自动约束在图片边界内 |
| **比例锁定** | 开启后，裁切框始终保持与网格一致的比例（例如 3:3 即 1:1） |
| **水印系统** | 文字水印，可改字体、粗细、位置；支持「无损反色」或「纯色+半透明」 |
| **右下角死锁** | 开启后，水印自动固定在裁切框右下角，不可移动，适合固定品牌标识 |
| **高清导出** | 基于原图分辨率切片，输出 `.jpg` 压缩包，每片不低于原画质 |

---

## 如何使用

1. **上传图片** – 点击中央区域或拖拽图片
2. **调整裁切框** – 拖拽边缘/角点，或点击底部预设按钮
3. **添加水印（可选）** – 点击「＋ 添加水印」，右侧面板可精细调节
4. **导出切片** – 点击「保存超清图包」，自动下载 `SlicerPro_HD.zip`

> ⚠️ 注意：切片导出时会**真实渲染水印**（反色模式下无视觉污染，但叠加后仍藏于图内）

---

## 技术特性

- 基于 **Canvas + Fabric.js**，所见即所得
- 完整处理 **物理边界约束**（裁切框、水印均无法越界）
- 支持 **高分辨率导出**（非屏幕缩放，直接使用原图尺寸计算切片位置）
